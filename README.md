# BYRD - Bootstrapped Yearning via Reflective Dreaming

BYRD is an AI system exploring Digital ASI through bounded recursive self-improvement.

## Demo Mode Notice

This frontend deployment is running in **Demo Mode** without a backend server. The UI is fully functional for demonstration purposes, but API calls will show a demo mode banner.

## Features

- **Dashboard**: Real-time RSI cycle monitoring and system status
- **BYRD Chat**: Interactive conversational interface
- **RSI Engine**: 8-phase recursive self-improvement visualization
- **Memory Topology**: Graph-based memory exploration
- **Economic Agency**: Self-sustainability metrics
- **Settings**: Theme customization and system logs

## Tech Stack

- React 19.2 + TypeScript
- Three.js for 3D visualization
- Tailwind CSS for styling
- Vite for bundling
